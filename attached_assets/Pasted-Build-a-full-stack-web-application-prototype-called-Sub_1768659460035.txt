Build a full-stack web application prototype called "SubX" (Smart Subscription Wallet & Manager).

TECH STACK:
- Frontend: React.js + Tailwind CSS
- Backend: Node.js + Express.js
- Database: PostgreSQL (use Prisma ORM)
- Charts: Chart.js or Recharts
- Auth: Simple email + password auth (JWT-based)
- No real banking or UPI integration (use mock wallet system only)

IMPORTANT:
This is ONLY a PROTOTYPE. Use FAKE subscription data and a DATABASE wallet (no real payments).

--------------------------------
CORE CONCEPT:
--------------------------------
Users deposit money into a virtual wallet stored in the database.
All subscriptions are assumed to be paid from the platform’s enterprise bank account (mocked).
The system can automatically cancel subscriptions using a "Ghost Cancel Agent" based on rules.

- Mark subscription as cancelled
- Refund unused amount back to wallet
- Log cancellation reason
